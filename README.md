# Todo-list-html-css-js
A simple and interactive To-do list web application built using HTML, CSS, and JavaScript that allows users to add, delete, and manage daily tasks efficiently.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter Active and Completed tasks
- Clear completed tasks
- Shows remaining task count
- Data stored using Local Storage

## Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage (Browser Storage)

## Purpose of Project
This project is created to practice DOM manipulation, event handling, and Local Storage in JavaScript.
